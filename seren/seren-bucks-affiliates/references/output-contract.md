# Seren Bucks V1 Output Contract

Return the operator-facing digest first, then the structured payload.

Required payload fields:

- `run_status`
- `mode`
- `generated_at`
- `program`
- `tracked_link`
- `auth_path`
- `affiliate_feed_status`
- `database_status`
- `provider_health`
- `candidate_sync`
- `proposal_top10`
- `pending_approvals`
- `dnc_events`
- `daily_cap`
- `daily_digest`
- `audit`
